### Phase 2: Integrated Agents (Slides 7-9)
**"L'Agent" — GABI Inside ChatTCU**

**GABI = Gabinete Automatizado de Boas Ideias (Automated Office of Good Ideas)**
- INTEGRATED INTO ChatTCU (not separate platform)
- **Published in PursuIT 10th Edition 2025** by International Centre for Information Systems and Audit (India)
- **Authors:** Pedro C. Filho, Fernando Gama Jr, Klauss Nogueira (TCU)
- **Article Title:** "AI Applied to Public Auditing: The TCU's Experience with ChatTCU and GABI"

**GABI Functions:**
- Meeting transcription from audio/video
- Automatic minutes drafting
- YouTube video analysis
- Work paper generation
- Creates editable documents with participants, objectives, key points, conclusions

**Key Differentiator:** Proactive — executes tasks without being asked

**Slide 7:** From Chatbot to Agent  
**Slide 8:** GABI Meeting Intelligence  
**Slide 9:** GABI Impact & PursuIT Recognition

### Phase 3: Audit as Code (Slides 10-12)
**"Le Paradigme" — Where Audit is Planned and Executed Like Software Engineering**

> "Audit is NOT just an assistant, NOT just a code-maker, but from where Audit is PLANNED and EXECUTED like software engineering"

**Core Concepts:**
- Deterministic, versioned, testable
- Codified audit logic
- Git-tracked audits
- Infrastructure as Code meets Audit
- content_hash for version identity
- Append-only immutability
- Cryptographic commitment layer
- CRSS-1 (Canonical Registry Serialization Specification)
- **Programmatic LaTeX (.tex) file generation - audit specifications compile to publication-ready documents**

**The Auditor's New Role:** Architect + Validator (not just executor)

**Flow Diagram:**
```
Audit Code (YAML) → Processing → LaTeX (.tex) → PDF Reports
```

**Why LaTeX Output Matters:**
- **Version control for documents** — every change tracked in Git
- **Reproducible audit reports** — identical inputs produce identical outputs
- **French legal document tradition** — LaTeX widely used in French academic and legal circles for precise document preparation
- **Mathematical precision** — typesetting quality for complex formulas and tables

**French Connection:**
> "LaTeX widely used in French academic and legal circles for precise document preparation" — aligns with Cour des comptes' tradition of exactitude

**Slide 10:** The Paradigm Shift (Copernican Inversion)  
**Slide 11:** The Deterministic Registry (Cryptographic Memory)  
**Slide 12:** Software Engineering Meets Audit

## 🔧 GABICHECKS: THE SEPARATE TOOL (Slides 13-14)

**IMPORTANT DISTINCTION:**

| | GABI | GABICHECKS |
|---|---|---|
| **Purpose** | Meeting documentation | Document verification |
| **Input** | Audio, video, YouTube | PDF documents + JSON checklists |
| **Output** | Meeting minutes, transcripts | Compliance reports |
| **Integration** | Inside ChatTCU | Separate tool |
| **Publication** | PursuIT 2025 (India) | Acórdão 1372/2025, Apêndice I |
| **Phase** | Phase 2 | After Phase 3 |

**GabiChecks Function:**
- Automates tedious verification tasks using LLMs
- Processes PDF documents for compliance
- Extracts text, applies AI analysis against checklists
- Generates reports with evidence and justifications
- Human review still required

**Slide 13:** GabiChecks Document Verification  
**Slide 14:** GabiChecks Workflow & Practice
