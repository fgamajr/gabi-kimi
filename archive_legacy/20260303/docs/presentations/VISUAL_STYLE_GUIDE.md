#### Prohibited Combinations

- вқҢ Multiple bright colors competing (visual chaos)

## 2. TYPOGRAPHY

### 2.1 Headline Fonts вҖ” French Elegance

| Font Family | Weight | Size Range | Usage | Example |
|-------------|--------|------------|-------|---------|
| **Didot** | Regular, Bold | 48-72pt | Main titles, dramatic openings | "Trois Phases, Une RГ©volution" |
| **Garamond** | Regular, Italic | 36-48pt | Secondary headlines, French quotes | "L'avenir n'est pas Г  prГ©dire" |
| **Bodoni** | Bold | 42-60pt | Statistical callouts | "650,000+ messages" |
| **Playfair Display** | 700, 900 | 40-64pt | Alternative headline font (digital) | Title slides, major sections |

#### Headline Typography Rules
- **Titles**: ALL CAPS for dramatic emphasis, Title Case for formal presentation
- **French phrases**: Italic Garamond to signal cultural respect
- **Statistics**: Bold Didot/Bodoni with Gold (#C9A961) color
- **Line spacing**: 1.1-1.2 for headlines (tight but readable)
- **Letter spacing**: +2% tracking for ALL CAPS headlines

### 2.2 Body Fonts вҖ” Modern Readability

| Font Family | Weight | Size Range | Usage | Line Height |
|-------------|--------|------------|-------|-------------|
| **Avenir** | Light, Book, Medium | 18-24pt | Primary body text | 1.5-1.6 |
| **Helvetica Neue** | Light, Regular | 16-22pt | Data labels, technical content | 1.4-1.5 |
| **Source Sans Pro** | Regular, Semibold | 14-20pt | Charts, tables, infographics | 1.3-1.4 |
| **Open Sans** | Regular, Bold | 16-20pt | UI elements, buttons | 1.4 |

#### Body Typography Rules
- **Maximum line length**: 60-75 characters for optimal readability
- **Paragraph spacing**: 1.5x line height between paragraphs
- **Bullet points**: 20-22pt Avenir Medium with 1.5 line height
- **Table text**: 14-16pt Source Sans Pro for data density

#### LaTeX Typesetting Styling Rules
- **Source code blocks**: Dark background (Midnight #0A1628), Code Green syntax highlighting, Fira Code font
- **Document preview**: Light background (Warm White #FAFAF8), Computer Modern serif font, PDF Red accents for annotations
- **Math formulas**: Computer Modern math italic, LaTeX Blue (#008080) for equation numbers
- **File extensions**: `.tex` in LaTeX Blue, `.pdf` in PDF Red, `.yaml` in Code Green
- **Compilation flow**: Golden arrows (#C9A961) connecting source вҶ’ process вҶ’ output

### 6.2 Whitespace Usage

#### The Rule of Thirds (Whitespace Edition)
- **40% content** вҖ” text, images, data
- **35% whitespace** вҖ” breathing room, visual rest
- **25% structural elements** вҖ” margins, gutters, borders

#### Whitespace Guidelines
| Element | Minimum Space | Optimal Space |
|---------|---------------|---------------|
| Slide edge to content | 5% of width | 8-10% for elegance |
| Headline to body text | 1.5x line height | 2x line height |
| Paragraph spacing | 1x line height | 1.5x line height |
| Between sections | 2x content height | 3x for dramatic pause |
| Image to caption | 8px | 16px |

#### French Elegance Principle
> "Elegance is refusal." вҖ” Coco Chanel

- Resist filling every space
- Let important elements breathe
- Use negative space to signal sophistication

### 6.4 Animation Suggestions

#### Animation Philosophy
- **Purposeful**: Every animation serves communication
- **Restrained**: French elegance favors subtlety
- **Consistent**: Same animation type for same content type

#### Recommended Animations

| Content Type | Animation | Duration | Easing |
|--------------|-----------|----------|--------|
| **Slide transition** | Fade or push | 0.6s | Ease-in-out |
| **Headline reveal** | Fade + slight rise | 0.4s | Ease-out |
| **Body text** | Fade in | 0.3s | Linear |
| **Data points** | Sequential fade | 0.2s each | Ease-out |
| **Diagram elements** | Draw/build in sequence | 0.5s each | Ease-in-out |
| **Bridge/connection** | Line draw | 1.0s | Ease-in-out |

#### Animation Patterns

**Three Phases Reveal**:
1. Phase 1 fades in (left)
2. Golden connector draws (center)
3. Phase 2 fades in (center-right)
4. Golden connector draws (right)
5. Phase 3 fades in (right)

**Statistical Build**:
1. Container appears
2. First statistic counts up (0 вҶ’ final)
3. Second statistic counts up
4. Third statistic counts up
5. Context text fades in

**Quote Elegance**:
1. Background atmospheric image fades in
2. Quote text fades + slight scale (0.95 вҶ’ 1.0)
3. Attribution fades in below
4. Decorative element draws (optional)

#### Animation DON'Ts
- вқҢ No spinning elements (unprofessional)
- вқҢ No bouncing entrances (childish)
- вқҢ No sound effects (distracting)
- вқҢ No random movement (chaos)
- вқҢ No excessive duration (impatience)
