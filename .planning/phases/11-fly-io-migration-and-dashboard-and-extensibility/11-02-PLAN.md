Create `src/backend/worker/migration.py`:

1. `async def migrate_catalog_to_sqlite(catalog_path: str, registry: Registry, es_url: str) -> dict`:
   - Read `catalog_path` JSON file (expects `{"files": [...]}` with objects containing filename, year_month, section, folder_id)
   - Query ES at `{es_url}/gabi_documents_v1/_search` with aggregation on `year_month` + `section` fields to determine which (year_month, section) combos already have indexed docs
   - For each catalog entry:
     - If (year_month, section) exists in ES aggregation results -> status = INGESTED
     - Else -> status = DISCOVERED
