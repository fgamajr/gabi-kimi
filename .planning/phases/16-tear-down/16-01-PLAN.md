  <action>
Run `fly apps list` to get the definitive list of gabi-dou-* apps. For each app found, run `fly secrets list -a {app}` and save the output to `.env.fly-backup` (one section per app with a header comment). Also save the app names to a temp list for the destruction step.

The fly CLI is at `/home/parallels/.fly/bin/fly`.

IMPORTANT: Add `.env.fly-backup` to `.gitignore` if not already there. This file must NEVER be committed.

Expected apps (from deploy configs): gabi-dou-frontend, gabi-dou-web, gabi-dou-worker, gabi-dou-es, gabi-dou-db. There may also be a redis app.

Format of .env.fly-backup:
```
# Fly.io secrets backup - {date}
# App: gabi-dou-web
SECRET_NAME=<from fly secrets list>
...
```

Note: `fly secrets list` shows names but not values. For actual secret values, use `fly ssh console -a {app} -C "env"` if the app is still running, or check if values are already known from ops/deploy configs and .env files.
